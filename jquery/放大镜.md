  ###### 1.让遮罩层移动
     步骤： {
         1.按下遮罩层，获取遮罩层的父盒子距离视口左边的距离和鼠标点击遮罩层时的那个点距离遮罩层左边的距离。
         2.在页面中绑定移动事件，获取遮罩层距离父盒子的距离
         3.给遮罩层改变left和top的值
                  }
  ###### 2.让大图移动
      {   
        1.算比率：遮罩层移动的距离 /遮罩层最多可移动的距离(c,k)
        2.得出大图最大移动的距离(x,y)
        3.算大图left值:c*x;算大图top值:  k*y
        4.给大图改变left和top的值
      }
